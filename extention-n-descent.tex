\chapter{РАСШИРЕНИЕ КОЭФФИЦИЕНТОВ И СПУСК}

Пусть есть мультипликативное подмножество $f_1, \dots, f_n$ кольца $R$,
порождающих единичный идеал.
Введем $R' := \prod\limits_{i = 1}^{n} R_{f_i}$.
Покажем, что морфизм $R \To R'$ -- вполне плоский.

Распишем $R' = \prod\limits_{i = 1}^{n} R_{f_i} = R_{f_1} \times \dots \times R_{f_n}$.
Множество $f_1, \dots, f_n$  -- мультипликативное,
тогда по (\ref{mult-sys-n-canon-isomor})
$R \To R_{f_i}$ -- плоский гомоморфизм модулей ($i =\overline{1,n}$). Если рассмотреть $R'$ как прямую сумму $R$-модулей, тогда по (\ref{flatness-of-dir-sum}) $R'$ -- плоское кольцо. Кроме того, если есть максимальный идеал $\mathfrak{m} \subset R$ , то найдется такой $f_j \notin \mathfrak{m}$ ($j = \overline{1,n}$), что $\mathfrak{m}R_{f_i} \subsetneq R_{f_i}$. Таким образом, $R \To R'$ -- вполне плоский гомоморфизм по (\ref{faithfully-flatnes-n-max-ideal}).

Для тех же $R$ и $R'$ покажем, что $R\text{-}\mathbf{Mod}$ категория эквивалентна (декартовому) произведению категорий $\prod\limits_{i = 1}^{n} R_{f_i}\text{-}\mathbf{Mod}$.

Категория $K\text{-}\mathbf{Mod}$ $K$-модулей для произвольного кольца $K$ составляется следующим образом:
\begin{itemize}
    \item объектами категорий будут $K$-модули;
    \item стрелками -- гомоморфизмы модулей.
\end{itemize}

    \begin{itemize}
        \item $\Ob{\cat{E}} = \Ob{\cat{C}} \times \Ob{\cat{D}}$.\\
